hexo 常用指令

发表于 2017-12-12

本文主要总结下hexo博客 常用的命令
对于我这种很久写一篇博客的人来说 真的是很有必要的一件事
嗯…时间隔太久…连发布命令都不记得 = =

从安装开始

有哪些简写

hexo new “我的博客” === hexo n “我的博客” #新建文章
hexo publish === hexo p
hexo generate === hexo g #生成
hexo server === hexo s #启动服务器预览
hexo deploy === hexo d #部署

服务器

hexo server #监视文件变动并自动更新
hexo server -s #静态模式
hexo server -p 5000 #更改端口
hexo server -i xxx.xxx.xxx.xxx #自定义端口
hexo clean #清除缓存
hexo g #成成静态页面
hexo d #开始部署

完成后部署

hexo generate –deploy
hexo deploy –generate
hexo deploy -g
hexo server -g

总结固定套路

hexo new “postName” #新建文章
hexo new page “pageName” #新建页面
hexo generate #生成静态页面至public 目录
hexo server #开启预览访问端口
hexo deploy #将deploy目录部署到github
